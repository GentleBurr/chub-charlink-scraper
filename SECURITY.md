# Security Policy

## Supported Versions
Currently, only the latest version of the `main` branch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability
Since this is a client-side bookmarklet, standard server-side vulnerabilities do not apply. However, security is still a priority. 

If you find a way this script could be used maliciously or if it handles data in an unsafe way, please **do not open a public issue.** Instead, please report the vulnerability privately by messaging me on Reddit:
https://www.reddit.com/u/SirGentlenerd/

I will investigate and address all legitimate reports as quickly as possible.
